Introduction

This lesson is designed for graduate and advanced undergraduate students undertaking original research projects. By the end, you will be able to format raw data in Excel for efficient analysis, run descriptive statistics, and prepare your dataset for PASW.

Required Tools & Prerequisites

  • Microsoft Excel
  • PASW (formerly SPSS) or free trial download

Familiarity with:

  • Basic Excel functions
  • Descriptive and inferential statistics
  • Research design (independent vs dependent variables)

Step 1: Format Data for Analysis

Open the tutorial file ToolBox_DataLesson.xls in Excel and follow these steps:

  • Ensure each variable has a unique name in the first row.
  • Rename variables to be intuitive (e.g., use → perc_usefulness).
  • Make categorical variables dichotomous (0 = no, 1 = yes).
  • Separate compound columns into components if necessary (e.g., task1, task2, task3).
  • Double-check for data entry errors before proceeding.

Step 2: Run Descriptive Statistics in Excel

Use Excel’s Analysis ToolPak:

  1. Go to Tools > Add-Ins and enable Analysis ToolPak.
  2. Select Data Analysis > Descriptive Statistics.
  3. Enter your input range (continuous data only) and check Summary Statistics.
  4. Output results in a new sheet and rename it Descriptive_Stats.

Step 3: Run Inferential Statistics in PASW

  • Open PASW and select the Excel file as the data source.
  • Load the Formatted_Data worksheet.
  • Use Analyze menu to run tests (e.g., Kruskal-Wallis for perc_usefulness by tutorial type).
  • Export results to Word, Excel, or PDF.
