Getting Started

Accessing Library Services

If you are newly admitted and haven’t obtained an IIT Hawk Card, visit the Hawk Card Homepage. You’ll need your card number to access some library services.

You can still access online journals and perform searches through the library website without it — the card is only required for physical checkout or interlibrary loan requests.

Using myIIT to Access the Library Website

Log in to myIIT, click "Library Resources," and use the proxy server to access most journals without reentering login details.

Start with scholarly databases like Academic Search Premiere (EBSCO) for broad journal searches.

Research Timelines

Formulating a Timeline

Balance obtaining, reading, and integrating resources with your project deadlines.

Being Realistic

Don’t assume you can read a full scholarly book in a day. Only request what you can actually use.

Requesting Resources

Requesting Articles

When articles aren’t downloadable as PDFs, use MyILL to request them.

Requesting Books

Search IIT’s library first, then I-Share, then WorldCat if needed. Record each resource in a shared document for tracking.

Local Libraries

I-Share Libraries

Most I-Share libraries let you check out books with your IIT ID. Always confirm hours and access before visiting.

Public Libraries

Chicago Public Library offers materials outside I-Share. Visitors can copy or read in-house — proof of residency is required for borrowing.

Internet Resources

Using Web Resources for Research

  • Google Scholar: Search scholarly articles and citations
  • Google Books: Preview or read full books
  • Web of Science: Search scientific papers and references

Being Realistic

Try special issues or edited volumes when stuck. They provide focused snapshots of a field.

Conclusion

Starting early gives you the widest access to resources. Use the methods covered to plan, locate, and evaluate sources effectively.
